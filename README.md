# Poke-Idle Auto-Clicker

Extension Chrome qui automatise la gestion de la pension sur [Poke-Idle](https://poke-idle.fr). Elle détecte et clique automatiquement sur le bouton **"Récupérer les œufs"** puis sur **"Redéposer"** dans la modale d'éclosion, en boucle, à l'intervalle de ton choix.

---

## Installation

L'extension n'est pas publiée sur le Chrome Web Store. Elle s'installe manuellement en mode développeur, ce qui prend moins d'une minute.

### 1. Télécharger les fichiers

Clique sur **Code → Download ZIP** sur cette page GitHub, puis extrais l'archive. Tu obtiens un dossier `poke-idle-clicker/`.

### 2. Activer le mode développeur dans Chrome

Ouvre Chrome et va à l'adresse :

```
chrome://extensions/
```

En haut à droite, active le **Mode développeur**.

### 3. Charger l'extension

Clique sur **"Charger l'extension non empaquetée"** puis sélectionne le dossier `poke-idle-clicker/` (celui qui contient directement le fichier `manifest.json`).

L'extension apparaît dans la liste sous le nom **"Poke-Idle Auto-Clicker"**.

### 4. Épingler l'extension (recommandé)

Clique sur l'icône puzzle 🧩 dans la barre d'outils Chrome, repère **"Poke-Idle Auto-Clicker"** et clique sur 📌 pour l'épingler.

---

## Utilisation

1. Ouvre l'onglet **Poke-Idle** et navigue sur la page **Pension**.
2. Clique sur l'icône de l'extension dans la barre d'outils.
3. Règle l'intervalle de vérification (en secondes) selon tes préférences.
4. Clique sur **ON** pour activer — l'extension s'occupe du reste.
5. Clique sur **OFF** pour désactiver à tout moment.

L'extension n'est active que sur l'onglet où tu l'actives explicitement. Elle n'interfère pas avec les autres onglets.

---

## Mise à jour

Si une nouvelle version est disponible, télécharge le ZIP, remplace les fichiers dans le dossier, puis va sur `chrome://extensions/` et clique sur l'icône **🔄** sur la carte de l'extension.
